# BasicCustomHUDRemake

A custom HUD plugin for SCP: Secret Laboratory (SCP:SL).

## Requirements

- LabAPI - Auto Installed Via SCP:Secret Laboratory
- Hint Service Meow

## Features

- Customizable HUD via config
- Rules can be set and displayed
